demo_structured.html — Why "Just Return JSON" Isn't Enough

We'll send the same prompt to the AI 6 times and see how the output varies — then check each against a schema.
Prompt sent every time
Extract the product name, price (as a number), and category from this text. Return only JSON with fields: name, price, category.
Input: "The new UltraBoost running shoes are now on sale for £89.99 — perfect for athletes."
Our schema (what we require)
name: string — must be present
price: number — must be a numeric value, not text
category: string — must be one of: "Footwear", "Clothing", "Electronics", "Accessories"
format: response must be valid JSON only, no extra text
